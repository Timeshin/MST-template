# MST Template

#### Application: https://mst-template.vercel.app

## Work with project

### Install dependencies

`npm install`

### Start project in dev

`npm run start:dev`

### Start project in prod

```
  npm run build
  npm run start:prod
```

## Docker

### Run app in docker. DEV

`docker-compose up`

### Run and re-build container

`docker-compose up --build`

## Commits

### Before commits

`npm run prepare`
